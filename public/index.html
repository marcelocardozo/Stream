Streaming de Câmera via RTSP

Aguardando permissão da câmera...

Como acessar o stream RTSP:

  1. Use um player RTSP como VLC Media Player
  2. No VLC, vá em "Mídia" > "Abrir fluxo de rede"
  3. Cole a URL RTSP mostrada acima
  4. Clique em "Reproduzir"

Nota: Você precisa ter o FFmpeg instalado no servidor para que o streaming funcione corretamente.
